# 🐔 Lmar Vet ProFarm

## تطبيق إدارة مزارع الدواجن الذكي

تطبيق أندرويد متكامل لإدارة مزارع الدواجن اللاحم في الأنظمة المغلقة، يوفر حلاً شاملاً لتحسين الإنتاجية وتقليل الخسائر وضمان صحة القطيع.

---

## 📋 المعلومات الأساسية

- **اسم التطبيق:** Lmar Vet ProFarm
- **المطور:** د. ضيف الله الحسني
- **النسخة:** 1.0.0
- **منصة:** Android (API 28+)
- **اللغة:** Kotlin + Java

---

## ✨ الميزات الرئيسية

### 1. **إدارة المزرعة**
   - إضافة وتحديث بيانات المزارع
   - تتبع معلومات القطيع
   - إدارة دورات الإنتاج
   - تسجيل عمليات التغذية والسقاية

### 2. **التشخيص البيطري الذكي**
   - تشخيص أمراض الدواجن
   - تسجيل الأعراض والعلاجات
   - قاعدة معلومات طبية شاملة
   - تنبيهات صحية للقطيع

### 3. **التحليل الفني للأداء**
   - تحليل أداء القطيع
   - رسوم بيانية وإحصائيات
   - معدل النمو والوزن
   - معدل الإنتاجية

### 4. **إدارة التقارير**
   - تقارير يومية وشهرية
   - تقارير الصحة والعلاجات
   - تقارير الإنتاجية
   - تصدير التقارير (PDF/Excel)

### 5. **إدارة العاملين**
   - إدارة فريق المزرعة
   - تعيين المسؤوليات
   - تسجيل ساعات العمل

## 🚀 البدء السريع

### 1. المتطلبات المسبقة
```bash
# تأكد من تثبيت:
- Android Studio 4.2+
- JDK 11+
- Gradle 7.0+
```

### 2. استنساخ المشروع
```bash
git clone https://github.com/vetdaifhaidar-cmd/-.git
cd LmarVetProFarm
```

### 3. فتح المشروع
```bash
# افتح المشروع في Android Studio
# File > Open > اختر مجلد المشروع
```

### 4. تشغيل التطبيق
```bash
# اضغط على زر Run في Android Studio
# أو من خلال الأمر
./gradlew assembleDebug
```
